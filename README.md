# elin-mod-ci-infrastructure

[elin-mod-dev-setup](https://github.com/sk-0520/elin-mod-dev-setup) や [elin-mod-WidgetMinimapPlus](https://github.com/sk-0520/elin-mod-WidgetMinimapPlus) で使用している CI 用のライブラリ寄せ集めです。

色々あって黒よりかはグレーっぽくなった気がします。

以下のアセンブリはインターフェイスとして配置されているだけで、実装部分はスタブに差し替えられています。

* Elin.dll
* Plugins*.dll

型情報だけ参照する思想なので、該当アセンブリを使用するようなテストなどは動かないと考えてください。

本リポジトリは可能な限り、最新の Elin に追従していますが、努力目標なだけで、古い可能性があります。  
なお、追従するのは安定版です（安定版しかプレイしないので）
